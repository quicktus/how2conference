# how2conference
Ich habe im Sommersemester 2021 begleitend zum Modul Machine Learining for Business (ML4B) die _Machine Learning for Business Conference 2021_ organisiert. Das Projekt wurde im Format des Forschenden Lernens durchgeführt. Dieses Reposistory dient der Dokumentation meiner Arbeit und kann als Vorlage für zukünftige ML4B Konferenzen verwendet werden.
Die ML4B Conference 2021 war eine interne Konferenz auf der Studierende die das Modul ML4B belegt haben die Ergebnisse ihrer Forschungsprojekte vorgestellt haben. Dazu haben sie ein Paper eingereicht, Papers anderer Studierenden Peer-Reviewt und schließlich ihr Paper in Form einer Präsentation auf der Konferenz vorgestellt.
